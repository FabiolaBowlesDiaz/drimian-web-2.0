  <action>
1. Create the Astro 5 project in the CURRENT directory (drimian-web2.0 already exists with _reference_repo/ and .planning/). Run:
   ```
   pnpm create astro@latest . -- --template minimal --typescript strict
   ```
   If prompted about existing files, allow overwriting (the project scaffold should NOT touch .planning/ or _reference_repo/).

2. Add Preact integration:
   ```
   pnpm astro add preact
   ```

3. Install Tailwind CSS v4 via the Vite plugin (NOT @astrojs/tailwind which is deprecated for v4):
   ```
   pnpm add tailwindcss @tailwindcss/vite
   ```

4. Install dev tools:
   ```
   pnpm add -D @astrojs/check typescript
   ```

5. Configure astro.config.mjs:
   ```typescript
   import { defineConfig } from 'astro/config';
   import preact from '@astrojs/preact';
   import tailwindcss from '@tailwindcss/vite';

   export default defineConfig({
     output: 'static',
     site: 'https://drimian.com',
     integrations: [preact()],
     vite: {
       plugins: [tailwindcss()],
     },
   });
   ```

6. Create a minimal src/pages/index.astro that imports Base layout (placeholder for now -- just enough to verify the build works):
   ```astro
   ---
   import Base from '../layouts/Base.astro';
   ---
   <Base title="Drimian">
     <main>
       <h1>Drimian</h1>
       <p>Foundation build in progress.</p>
     </main>
   </Base>
   ```

CRITICAL ANTI-PATTERNS:
- Do NOT create tailwind.config.js (Tailwind v4 uses CSS-first @theme)
- Do NOT use `astro add tailwind` (installs deprecated @astrojs/tailwind)
- Do NOT add any `content` array -- Tailwind v4 auto-detects

   - Navigate to https://fonts.google.com/specimen/Playfair+Display
   - Download the font family, extract the relevant weights
   - Convert to WOFF2 if needed (Google Fonts provides TTF by default)

   Alternative approach using the Google Fonts CSS API to get WOFF2 URLs directly:
   ```bash
   # Get WOFF2 URLs by requesting with a modern user-agent
   curl -s -H "User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36" \
     "https://fonts.googleapis.com/css2?family=Playfair+Display:wght@600;700&display=swap&subset=latin" \
     | grep -oP 'url\(\K[^)]+\.woff2'
   ```
   Download each URL and save to public/fonts/.

3. Download Inter variable font (Latin subset, weights 400-600):

   ```bash
   curl -s -H "User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36" \

     | grep -oP 'url\(\K[^)]+\.woff2'
   ```
   Download and save as public/fonts/inter-latin-var.woff2.
